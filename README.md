# PWNBot
Simple bot in python that will automatically complete PWN form.

# Usage Windows
git clone https://github.com/Czer0xx/PWNBot  <br/>
pip install -r requirements.txt <br/>
python pwnbot.py

# Usage Linux
git clone https://github.com/Czer0xx/PWNBot <br/>
pip3 install -r requirements.txt <br/>
python3 pwnbot.py
